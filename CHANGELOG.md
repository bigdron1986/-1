# Журнал изменений — Термометрия

## Версия 1.2.0 (март 2026)

### Новые функции

#### 1. Всплывающие подсказки в таблице "Горячие точки"
- При наведении курсора на ячейку таблицы отображается:
  - Информация о датчике (Силос, Подвеска, Датчик)
  - Текущая температура
  - Дельта за сутки (разница с предыдущей датой)
  - Предыдущая температура

#### 2. Отслеживание смены лидера (самого горячего датчика)
- Под таблицей "Горячие точки" отображается информация о самом горячем датчике
- Если лидер изменился, показывается:
  - Предыдущий лидер (был: X)
  - Текущий лидер (стал: Y)
  - Дельта температур между лидерами

#### 3. Погодный виджет
- Отображение температуры и влажности на улице
- Данные загружаются с OpenWeatherMap
- Координаты: 56.1345982, 69.4616057 (Курганская область)
- Время обновления: при запуске приложения и при загрузке нового отчета
- Цветовая индикация температуры:
  - < 0°C: голубой (мороз)
  - 0-10°C: светло-голубой (холодно)
  - 10-20°C: зелёный (комфортно)
  - 20-30°C: жёлтый (тепло)
  - > 30°C: красный (жарко)

### Изменения в базе данных

Добавлены новые функции в `database.py`:
- `get_hottest_sensor_for_date()` — получить самый горячий датчик за дату
- `get_previous_date_with_data()` — получить предыдущую дату с данными
- `get_sensor_temperature_on_date()` — получить температуру датчика на дату
- `get_leader_change_info()` — получить информацию о смене лидера

### Новые файлы

- `weather.py` — модуль для работы с погодным API
- `WEATHER_SETUP.md` — инструкция по настройке API ключа

### Изменения в конфигурации

В `app_config.json` добавлены поля:
- `openweathermap_api_key` — API ключ OpenWeatherMap
- `weather_last_data` — последние успешные погодные данные

### Настройка погоды

1. Получите бесплатный API ключ на [openweathermap.org](https://home.openweathermap.org/api_keys)
2. В приложении нажмите кнопку "⚙️" в панели погоды
3. Введите API ключ и сохраните

### Совместимость

- Python 3.13+
- Все зависимости из `requirements.txt`
- **Новые зависимости не требуются** (weather.py использует встроенные модули)

### Известные ограничения

- Погодный виджет требует API ключ OpenWeatherMap (бесплатно)
- Лимит API: 60 запросов/мин, 1 млн/мес (более чем достаточно)
- При отсутствии ключа показывается "Н/Д"
